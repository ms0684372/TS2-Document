<a id="dungeon"></a>
*   ## 副本
    ### 說明：主要是打小布包和裝備用的，打叉的是幾乎沒人打，前面10~20等 (奪糧、包逆、蘋果) 可打可不打，因為裝備很快就汰換掉了
    ### 各副本說明：
    ### 奪糧的試煉(Lv10)：奪糧、9包 (9包由來是一場最多可以拿9個土黃布包)
    ### 包子的逆襲(Lv10)：包逆
    ### 長生不老(Lv10)：
    ### 保衛樓桑村(Lv20)：保衛村長、村長 (村長真的有夠爛)
    ### 青蘋果之役(Lv20)：青蘋果、蘋果
    ### 決戰泰山寇(Lv30)：泰山
    ### 淄博撤退戰(Lv30)：撤退
    ### 懸壺濟世(Lv40)：葫蘆
    ### 黃金風雲起(Lv40)：40黃巾
    ### 百姓的哀鳴(Lv45)：百姓
    ### 黃巾初章河南激戰(Lv45)：45黃巾、河南激戰
